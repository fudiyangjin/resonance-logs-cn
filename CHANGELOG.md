# Changelog v0.2.0-beta

## 更新内容

### 语音播报（新增）

- 新增离线语音播报：安装模型后可克隆音色、维护短句库，并在触发时播放已生成音频
- 支持在 Buff 监控、怪物 Buff、自定义计数器、Boss DBM、副本机制中内联配置语音绑定
- 播报总览页可汇总全部绑定，并一键生成缺失语音
- 支持 CPU / Vulkan 加速与播放队列策略（丢弃低优先级 / 高优先级打断）
- Buff / 怪物 Buff「生效时 / 失效时」自定义文本支持 `${阶数}`（或 `${remodelLevel}`）占位符，按幻想 0～5 阶按需生成对应语音
- 使用说明见 [语音播报指导手册](./doc/zh-CN/features/voice/README.md)

### 监控方案

- 总方案（Loadout）能力完善：组合技能监控、怪物监控与实时计量子方案，一键切换整套设置
- DPS / 实时计量相关设置拆分并纳入方案控制，可随总方案导出分享
- 新增光盾职业预置方案；首启可选预置方案快速开始
- 支持方案导入 / 导出

### DPS / 历史

- 死亡回放增加受击参与者的 Buff 快照展示 @Sanheiii
- 统一背景元素设置与提示框样式

### 实时监控相关

- 支持自定义 Buff 图标（可对任意 Buff 设置图片）
- Buff 监控支持配置语音播报（生效 / 失效 / 提前 N 秒等）

### 怪物监控相关

- Boss DBM、怪物 Buff 支持配置语音播报
- Buff 名称文档调整

### 副本机制相关

- 支持机制语音播报绑定（各适配地图内可配置）
- 支持自定义阵亡样式（颜色、透明度、× 标记）
- 机制语音播报配置区默认收起

### 数据与本地化

- 正式服 1.2.43804.0 数据更新 @Yozora_Hoshi
- 补充日文 / 英文数据，补全幻想技能

## 注意事项

### 兼容性提醒

- 如果之前使用过 0.0.2 或 0.0.3，第一次使用 0.0.4-0.2.0-beta 需要清理 `%LOCALAPPDATA%\resonance-logs-cn` 目录下的 `resonance-logs-cn.db`，再重新启动应用
- 0.1.5 由于修改了实例存储 key（uid -> uuid）、抓包逻辑，整体改动较大，若有异常请及时联系并提供日志
- 0.1.5 新增的因子有些没有进行自测（因子太贵了，并且没时间），如果有异常请及时反馈
- 语音播报需单独下载模型；首次生成短句可能耗时较长，生成完成后播放不再加载模型

### 特别注意

- 如果你自己修改了代码又不想提交 PR，在分享给其他人的时候请把应用名、版本号等原版相关信息都改掉，不要带着原仓库的信息，尤其是应用名和版本号
- 关闭按钮现在会隐藏到右下角，可以将隐藏栏的内容拖到右下角显示，避免窗口程序过多时无法快速找到
- 常见问题见使用手册中的 HTML
