# Changelog v0.1.6

## 更新内容

### DPS 相关

- WinDivert 由于用的人还是比较多, 因此重新加回来了

### 实时监控相关

- 移除编辑模式的幽灵层展示, 直接显示另一层. 比如编辑怪物监控时显示实时监控的已布局情况
- 同步怪物监控, 实时监控编辑模式的大小, 位置
- 血量护盾区样式自定义
- 修复冰法灵感 X2 因子, 同时暴击幸运触发的陨石没有正常计数的问题
- 修复误置了多份 S2 因子 itemid 导致的计数异常
- 自定义面板新增设置计数为 0 时是否显示
- 增加真实因子上限映射, 因子区计数显示 "当前/上限"
- 重构角色技能释放模型, 会影响到涉及技能释放的功能
- 炎角监控新增射线相关技能

### 怪物监控相关

- 修复队友 buff 队友过多时导致的样式异常, 增加更多的样式设置
- 修复切图后队友 buff 消失的问题
- 新增展示队友使用幻想

## 注意事项

### 兼容性提醒

- 如果之前使用过 0.0.2 或 0.0.3，第一次使用 0.0.4-0.1.6 需要清理 `%LOCALAPPDATA%\resonance-logs-cn` 目录下的 `resonance-logs-cn.db`，再重新启动应用
- 0.1.5 由于修改了实例存储 key(uid->uuid), 抓包逻辑, 整体改动较大, 若有异常请及时联系并提供日志
- 0.1.5 新增的因子有些没有进行自测(因子太贵了, 并且没时间), 如果有异常请及时反馈

### 特别注意

- 如果你自己修改了代码又不想提交 PR，在分享给其他人的时候请把应用名、版本号等原版相关信息都改掉，不要带着原仓库的信息，尤其是应用名和版本号
- 关闭按钮现在会隐藏到右下角, 可以将隐藏栏的内容拖到右下角显示, 避免窗口程序过多时无法快速找到
- 常见问题见使用手册中的 hmtl
