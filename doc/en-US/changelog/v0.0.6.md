# Changelog v0.0.6

## Logs

**DPS-related**

- True DPS now uses global active time instead of per-entity calculation
- Setting to show global active combat time (DPS Meter → Themes → Live → Timer → Show active combat time)
- History over 200 entries auto-deleted by time on next launch, sequence reset
- Fixed history overview vs detail time mismatch—unified to first/last combat packet delta

**Buff-related**

- Reorganized settings: Skill CD, Buff Monitor, Character Panel, Custom Panel, Enable Overlay
- Global buff aliases—search buff and rename (e.g. `[热枕]` → Life Fluctuation). Live Monitor → Buff Monitor → Buff Alias Settings; global, not profile-specific
- Standalone **Category Quick Listen** for all food or alchemy buffs with unified layout; grouped mode Food/Alchemy shortcut buttons
- Custom panel for counters and buff monitoring as progress bars; row spacing, font size, etc.; hide icons by showing buffs here
- No-icon buff display unified in one row with progress bar styling; legacy format supported
- Buff-linked damage-type state machine for special counters (e.g. Fantasy Impact, Transcend triggers). Live Monitor → Custom Panel → Add Counter
- Buff duration format: grouped and hour display
- Shortcut for **Toggle Overlay** (DPS Meter → Settings → Shortcuts)

**Panel attributes**

- Cooldown reduction and cooldown haste display

**Notes**

- If you used 0.0.2/0.0.3, first launch on 0.0.6 requires deleting `resonance-logs-cn.db` under `%LOCALAPPDATA%\resonance-logs-cn` and restarting
