# Settings

Corresponds to **DPS Meter → Settings**.

## Settings paths

| Setting | Path |
|--------|------|
| Player columns | DPS Meter → Settings → Live → DPS (Player) Columns |
| Skill columns | DPS Meter → Settings → Live → DPS (Skill Details) Columns |
| Refresh rate | DPS Meter → Settings → Live → Refresh Rate |
| Abbreviate DPS | DPS Meter → Settings → Live → Shorten DPS Values |
| Training dummy button | DPS Meter → Themes → Live → Header Settings → Show Training Dummy Button |
| Active combat time | DPS Meter → Themes → Live → Header Settings → Show Active Combat Time |
| Compact mode | DPS Meter → Themes → General → Compact Mode |
| Title bar free layout | DPS Meter → Themes → Live → Header Settings |
| Boss health | DPS Meter → Themes → Live → Header Settings → Boss Health |
| Packet capture | DPS Meter → Settings → Network |
| Shortcuts | DPS Meter → Settings → Shortcuts |

See [Themes](./themes.md) for appearance options. See [DPS Overview](./README.md#training-dummy-mode) for training dummy mode.
