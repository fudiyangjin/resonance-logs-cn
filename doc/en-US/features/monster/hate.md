# Threat List

Corresponds to **Monster Monitor → Hate List** tab.

Shows **threat order** for your **current main attack target** in the overlay (who is on the threat table and relative order)—updates with target switch; useful for tank threat and overtaunt checks.

## Enable Threat List

- Turn on **Enable Threat List** for threat data to parse and show in the overlay.
- If disabled, **Threat List Area** in [Enable Overlay](./overlay.md) cannot be enabled (enable here first).

## Display Count

**Max characters shown**: 5–20—caps threat entries in the overlay (default ~5).

## Threat Panel Style

Configure **Threat List Area** separately from monster buff and teammate buff areas:

- Row spacing, column spacing, font size
- Name / value / bar colors and opacity
