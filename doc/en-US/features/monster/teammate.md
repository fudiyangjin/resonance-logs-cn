# Teammate Buffs

Corresponds to **Monster Monitor → Teammate Buffs** tab.

Shows buffs **teammates applied to your current main attack target** (vs “self-only” on the Buff Monitor tab). Updates when main target changes—useful for raid buffs, teammate debuffs on Boss or dummy.

## How to Add

### Search Single Buff

Search by name, pick from results to add to **Monitored Teammate Buffs**.

### Category Shortcuts

**Category buttons** matching the buff catalog (food, alchemy, etc.—as shown in UI)—add whole category at once, or click selected category tags to remove.

## Monitored List

Selected single buffs and categories appear together; click tags to remove.

## Teammate Buff Matrix Style

Separate from monster buff area styling—controls **Teammate Buff Area** overlay:

- Row spacing, column spacing, font size
- Name / value / bar colors and opacity

Toggle **Teammate Buff Area** in [Enable Overlay](./overlay.md).
