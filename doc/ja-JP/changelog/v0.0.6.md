# Changelog v0.0.6

## Logs

**dps 関連**

- 真秒間ダメージの設定ロジックを変更。各エンティティ個別計算からグローバル活動時間ベースに
- グローバル活動時間の表示オン/オフ設定を追加（DPSメーター-テーマ-リアルタイム-タイマー-アクティブ戦闘時間を表示）
- 履歴が 200 件超の場合、次回起動時に時間順で自動削除しシーケンスをリセット
- 履歴総覧と詳細の時間表示不一致を修正。初回・最終戦闘パケット差分で統一

**buff 関連**

- 設定画面を再構成：スキル CD、Buff監視、キャラクターパネル、カスタムパネル、ウィンドウを有効化
- Buff グローバルエイリアス設定を追加。例：`[热枕]` を生命波动 に変更（ライブモニター-Buff監視-Buffエイリアス設定。グローバル、プリセット切替で変わらない）
- 独立モードに「カテゴリクイックモニター」を追加。食物・炼金 Buff を一括監視しレイアウト統一。グループモードに「食物」「炼金」ショートカット（ライブモニター-Buff監視-カテゴリクイックモニター）
- カウンターと Buff 監視用カスタムパネルを追加。プログレスバー表示。行間・フォントなど設定可。アイコン不要な Buff はここに配置可
- 无图标 Buff 表示を変更。同一行に統一表示。行間など設定可。旧形式に戻す設定可
- Buff 関連・ダメージタイプのステートマシンを導入。Buff で維持されない特殊ダメージ累計トリガーをカウント（例：幻想冲击、超然触发）（ライブモニター-カスタムパネル-カウンターを追加）
- Buff 持続時間形式にグループ・時間表示を追加
- 「オーバーレイウィンドウの切替」ショートカット設定をサポート（DPSメーター-設定-ショートカット）

**面板属性関連**

- 冷却缩减・冷却加速属性の表示をサポート

**注意**

- 以前 0.0.2/0.0.3 を使用していた場合、初めて 0.0.6 を使うときは `%LOCALAPPDATA%\resonance-logs-cn` 内の `resonance-logs-cn.db` を削除してから再起動

**特に注意**

コードを改変して PR しない場合、共有時はアプリ名・バージョンなど原版情報をすべて変更し、原リポジトリ情報（特にアプリ名+バージョン）を残さないでください。
