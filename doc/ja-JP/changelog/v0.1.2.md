# Changelog v0.1.2

## 更新内容

### DPS 関連

- damageid を追加 @YozoraHoshi
- 従来の閉じるボタンのロジックを変更。右下に非表示化。非表示バーの内容を右下にドラッグして表示可能。ウィンドウが多いときに素早く見つけられない問題を回避
- タイトルバーのボス HP に横並び表示オプションを追加（DPSメーター-テーマ-リアルタイム-ヘッダー設定-首領血量）
- タイトルバーレイアウトに自由配置を追加。各エリアの配置を個別に制御可能（DPSメーター-テーマ-リアルタイム-ヘッダー設定）
- コンパクトモード に戦力・シーズン強度の表示を追加（要設定）。live 画面で総ダメ・秒ダメ・占有率のみ表示する場合は有効化を推奨
  - playerA-職業 (戦力-シーズン強度) &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 総ダメ(秒ダメ/真秒ダメ) 占有率
  - playerB-職業 (戦力-シーズン強度) &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 総ダメ(秒ダメ/真秒ダメ) 占有率
  - playerC-職業 (戦力-シーズン強度) &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 総ダメ(秒ダメ/真秒ダメ) 占有率
- クリックスルー設定後、アプリ再開時に DPS 画面へ切り替えないと自動適用されない問題を修正
- クリックスルーショートカットが正常に永続化されない問題を修正

### ライブモニター関連

- シールド buff 詳細パネルを新規追加（ライブモニター-ウィンドウを有効化-血量护盾区） @18870
- 赤红 CD を追加
- 岩之心天賦カウンタが岩弹怒击天賦・無料の巨岩之怒で異常リセットされる問題を修正
- ワンクリック薬剤監視に耐性・增效薬を追加
- 真实因子を新規追加
  - 狼弓灵感因子 x1, x3, x4、真实因子 x1, x2

  - 冰法真实因子 x2
- マップ切替時に保存属性をクリア。全量同期で属性が同期されず古い値が表示され続けるのを防止

## 注意事項

### 互換性に関する注意

- 0.0.2 または 0.0.3 を使用していた場合、0.0.4〜0.1.2 を初めて使う際は `%LOCALAPPDATA%\resonance-logs-cn` 配下の `resonance-logs-cn.db` を削除してからアプリを再起動してください
- npcap の処理ロジックを最適化済みのため、パケットキャプチャには npcap を優先してください。WinDivert ドライバの署名期限切れにより、ウイルス対策ソフトの誤検知・誤隔離が発生し、アプリが正常に動作しなくなる可能性があります

### 特に重要

- コードを自分で改変し PR を出さずに他人に配布する場合は、アプリ名・バージョン番号など原版の情報をすべて変更し、特にアプリ名とバージョン番号を含めて原リポジトリの情報を残さないでください
- 閉じるボタンは右下に非表示化されます。非表示バーの内容を右下にドラッグして表示できます。ウィンドウが多いときに素早く見つけられない問題を回避できます
- よくある質問は使用マニュアルの HTML を参照してください
