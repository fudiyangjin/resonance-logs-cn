- 猛击追击・岩怒追击天賦カウンタを追加

### モンスターモニター関連

- buff アラートを新規追加。残り x 秒で色変化と点滅
- sdm ボス階で モンスターモニター が正常に動作しない問題を修正

## 注意事項

### 互換性に関する注意

- 0.0.2 または 0.0.3 を使用していた場合、0.0.4〜0.1.2 を初めて使う際は `%LOCALAPPDATA%\resonance-logs-cn` 配下の `resonance-logs-cn.db` を削除してからアプリを再起動してください
- npcap の処理ロジックを最適化済みのため、パケットキャプチャには npcap を優先してください。WinDivert ドライバの署名期限切れにより、ウイルス対策ソフトの誤検知・誤隔離が発生し、アプリが正常に動作しなくなる可能性があります

### 特に重要

- コードを自分で改変し PR を出さずに他人に配布する場合は、アプリ名・バージョン番号など原版の情報をすべて変更し、特にアプリ名とバージョン番号を含めて原リポジトリの情報を残さないでください
- 閉じるボタンは右下に非表示化されます。非表示バーの内容を右下にドラッグして表示できます。ウィンドウが多いときに素早く見つけられない問題を回避できます
- よくある質問は使用マニュアルの HTML を参照してください
