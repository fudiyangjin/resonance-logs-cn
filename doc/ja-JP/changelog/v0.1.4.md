# Changelog v0.1.4

## 更新内容

### DPS 関連

- 死亡リプレイに攻撃者表示を追加
- 一部流派判定スキルを変更
  - 居合は特攻のみで識別、月刃は神罚のみで識別
  - 冰矛は特攻のみで識別
  - 格挡・岩盾は特攻のみで識別
- マップ切替後にデータが表示されない可能性がある問題の修正を試行。更新後も npcap 使用時に問題が続く場合はログを提供してください

### ライブモニター関連

- 协奏スキル CD を追加
- カスタムパネルの設定読み込みシリアライズでエラーが出る場合がある問題を修正
- 空枪灵感因子 X7, X8, X9、真实因子 X4, X5 を追加
- 冰法灵感因子 X2 を追加
- 光盾・防盾の神盾裁决追击天賦カウンタを追加

## 注意事項

### 互換性に関する注意

- 0.0.2 または 0.0.3 を使用していた場合、0.0.4〜0.1.4 を初めて使う際は `%LOCALAPPDATA%\resonance-logs-cn` 配下の `resonance-logs-cn.db` を削除してからアプリを再起動してください
- npcap の処理ロジックを最適化済みのため、パケットキャプチャには npcap を優先してください。WinDivert ドライバの署名期限切れにより、ウイルス対策ソフトの誤検知・誤隔離が発生し、アプリが正常に動作しなくなる可能性があります

### 特に重要

- コードを自分で改変し PR を出さずに他人に配布する場合は、アプリ名・バージョン番号など原版の情報をすべて変更し、特にアプリ名とバージョン番号を含めて原リポジトリの情報を残さないでください
- 閉じるボタンは右下に非表示化されます。非表示バーの内容を右下にドラッグして表示できます。ウィンドウが多いときに素早く見つけられない問題を回避できます
- よくある質問は使用マニュアルの HTML を参照してください
