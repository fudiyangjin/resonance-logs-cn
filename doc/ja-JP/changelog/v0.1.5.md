# Changelog v0.1.5

## 更新内容

### DPS 関連

- アプリ全体の遅延応答を最適化
- WinDivert ロジックを削除し、パケットキャプチャ処理をリファクタリング
- ラッキー率の計算と表示を修正
  - スキルごとの発動ラッキー率を個別に集計
  - ラッキー率計算では「幸运一击」の命中数を除外
- Tank に格挡率、幸运格挡率の表示を追加
- Tank のリアルタイム・履歴記録は先にモンスター別で集約表示し、クリック後に該当モンスターのスキル被ダメージ詳細を表示
- 木人モードのロジックを変更
  - フロントエンド側の目標設定を削除し、木人モード開始後に精英攻击木桩または守护木桩を自動識別
  - フローを `待機 -> 実行中 -> 終了` に変更し、リセット後に待機へ戻る
  - 木人終了後、live 画面は更新されず、そのまま結果を確認可能
- 妖尾バージョンまでの正式サーバーデータを追加。buff、シーン、新規幻想などを含みます。@YozoraHoshi に感謝

### ライブモニター関連

### モンスターモニター関連

- `ExtraBuffMonitoredMonsters.json` 設定を廃止。モンスター buff とヘイトは、自分が攻撃している主目標を自動表示
- 味方 buff モニターを追加
- ライブモニターのゴーストレイヤー表示を最適化
- モンスター buff に「自分のみが付与」表示を追加し、監視対象 buff を探しやすく改善

## 注意事項

### 互換性に関する注意

- 0.0.2 または 0.0.3 を使用していた場合、0.0.4〜0.1.4 を初めて使う際は `%LOCALAPPDATA%\resonance-logs-cn` 配下の `resonance-logs-cn.db` を削除してからアプリを再起動してください
- 0.1.5 では WinDivert パケットキャプチャを削除しました。以前 WinDivert を使用していた場合は Npcap をインストールしてください
- 0.1.5 ではインスタンス保存 key（uid -> uuid）とパケットキャプチャ処理を変更しているため、異常があればログを添えて報告してください
- 0.1.5 では統計フィールドを追加したため、古い履歴記録は表示互換がありません
- 新規追加因子の一部は十分に自測できていないため、異常があれば報告してください

### 特に重要

- コードを自分で改変し PR を出さずに配布する場合は、アプリ名・バージョン番号など原版の情報を変更してください
- 閉じるボタンは右下に非表示化されます。非表示バーの内容を右下へドラッグして表示できます
- よくある質問は使用マニュアルの HTML を参照してください
