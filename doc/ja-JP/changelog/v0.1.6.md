# Changelog v0.1.6

## 更新内容

### DPS 関連

- 利用者が多いため WinDivert を再度追加

### ライブモニター関連

- 編集モードのゴーストレイヤー表示を削除し、別レイヤーを直接表示（例：モンスターモニター編集時にライブモニターのレイアウトを表示）
- モンスターモニターとライブモニターの編集モードのサイズ・位置を同期
- HP/シールドエリアのスタイルをカスタマイズ可能に
- 冰法 灵感 X2 因子で、会心とラッキーが同時発動した隕石が正常にカウントされない問題を修正
- S2 因子 itemid が重複設定されていたことによるカウント異常を修正
- カスタムパネルに、カウントが 0 のときに表示するかどうかの設定を追加
- 真实因子の上限マッピングを追加し、因子区のカウントを「現在/上限」形式で表示
- キャラクターのスキル発動モデルをリファクタリング。スキル発動に関わる機能に影響
- 炎角モニターにレイ関連スキルを追加

### モンスターモニター関連

- 味方 buff が多すぎる場合のスタイル異常を修正し、スタイル設定を追加
- マップ切り替え後に味方 buff が消える問題を修正
- 味方の幻想使用状況の表示を追加

## 注意事項

### 互換性に関する注意

- 0.0.2 または 0.0.3 を使用していた場合、0.0.4〜0.1.6 を初めて使う際は `%LOCALAPPDATA%\resonance-logs-cn` 配下の `resonance-logs-cn.db` を削除してからアプリを再起動してください
- 0.1.5 ではインスタンス保存 key（uid -> uuid）とパケットキャプチャ処理を変更しているため、異常があればログを添えて報告してください
- 0.1.5 で追加した因子の一部は十分に自測できていないため、異常があれば報告してください

### 特に重要

- コードを自分で改変し PR を出さずに配布する場合は、アプリ名・バージョン番号など原版の情報を変更してください
- 閉じるボタンは右下に非表示化されます。非表示バーの内容を右下へドラッグして表示できます
- よくある質問は使用マニュアルの HTML を参照してください
