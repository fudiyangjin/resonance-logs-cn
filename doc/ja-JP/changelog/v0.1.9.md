# Changelog v0.1.9

## 更新内容

### DPS 関連

- 言語選択をメインウィンドウ左下に移動
- ウィンドウ切り替えボタンが最小化状態に応じて表示を変更
- Reset Windows に他のオーバーレイウィンドウ位置のリセットを追加

### ライブモニター関連

- buff 検索で buff ID による検索をサポート @dilone
- 各機能エリアに任意表示の黒いオーバーレイを追加。デフォルトはオフ
- フォントシャドウのオン/オフを制御する設定を追加
- 既存のフォント設定がライブモニターオーバーレイの表示フォントにも適用されるように拡張（DPS 検出 → テーマ → 一般 → カスタムフォント → オーバーレイに適用）
- 惩击リソースを追加。2202251 を特殊 buff 表示に拡張。スキルが使用可能かどうかを対応リソースエネルギーと連動
- ウィンドウ切り替えボタンが最小化状態に応じて表示を変更

### モンスターモニター関連

- N20 ティナ攻撃低下イベント CD を追加
- 靭性バー表示を追加。モンスター buff やヘイトと同様、現在のメインターゲットを表示（モンスターモニター → 靭性バー）
- 各機能エリアに任意表示の黒いオーバーレイを追加。デフォルトはオフ
- フォントシャドウのオン/オフを制御する設定を追加
- 既存のフォント設定がモンスターモニターオーバーレイの表示フォントにも適用されるように拡張（DPS 検出 → テーマ → 一般 → カスタムフォント → オーバーレイに適用）
- Boss DBM にスキル名変更設定を追加
- ウィンドウ切り替えボタンが最小化状態に応じて表示を変更

### ダンジョンギミック関連

- ナボ、巨塔マップを調整
- 実際の攻撃範囲に合わせて波幅を調整し、事前被弾判定で表示（危険は赤、安全は緑）
- ミニマップ上のエンティティサイズを変更可能に
- 既存のフォント設定がダンジョンギミックオーバーレイの表示フォントにも適用されるように拡張（DPS 検出 → テーマ → 一般 → カスタムフォント → オーバーレイに適用）
- ウィンドウ切り替えボタンが最小化状態に応じて表示を変更

## 注意事項

### 互換性に関する注意

- 0.0.2 または 0.0.3 を使用していた場合、0.0.4〜0.1.9 を初めて使う際は `%LOCALAPPDATA%\resonance-logs-cn` 配下の `resonance-logs-cn.db` を削除してからアプリを再起動してください
- 0.1.5 ではインスタンス保存 key（uid -> uuid）とパケットキャプチャ処理を変更しているため、異常があればログを添えて報告してください
- 0.1.5 で追加した因子の一部は十分に自測できていないため、異常があれば報告してください

### 特に重要

- コードを自分で改変し PR を出さずに配布する場合は、アプリ名・バージョン番号など原版の情報を変更してください
- 閉じるボタンは右下に非表示化されます。非表示バーの内容を右下へドラッグして表示できます
- よくある質問は使用マニュアルの HTML を参照してください
