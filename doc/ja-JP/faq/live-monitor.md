## Buff / スキル CD

### 途中でアプリを開くと CD やステータスが欠けるのはなぜ？

ゲーム進行中にアプリを起動すると、スキル CD、Buff、キャラクターパネルの属性などが不完全に見えることがあります。ゲームは**差分更新**のみ送るため、アプリは現在の完全な状態を取得できません。**マップ切り替え**（ダンジョン出入り、エリア変更、テレポートなど）で一局分の属性がフル同期されると表示が正常に戻ります。

### Buff エイリアスは何に使いますか？

ゲーム内の一部 Buff 名は分かりにくい（例：`[热枕]`）ため、**リアルタイム監視 → Buff監視 → Buffエイリアス設定** で分かりやすい名前（例：「生命波动」）に変更できます。設定はグローバルで、プリセット切り替えでは変わりません。

### 真实因子（リアル因子）とは？どこで設定？

「真实因子」は、対応するシーズン育成因子を現在持っているか（多くは `0` / `1`）を示します。**リアルタイム監視 → カスタム監視 → 因子区を新規作成** で因子エネルギーと Buff 効果を自動表示するのがおすすめで、カウンタールールを手動で組む必要はありません。

- **因子エネルギーなどのカウンター名**：因子区の **因子表示名** で検索・設定。
- **因子 Buff 効果の名称**：[Buff監視 → Buffエイリアス設定](../features/monitor/buff.md#buff-エイリアス) で設定。

詳細は [カスタム監視 · 因子区](../features/monitor/custom-panel.md#因子区因子区を新規作成) を参照。

### 食事 / 錬金 Buff を一括で監視するには？

- **独立モード**：「分類ショートカット監視」で食事または錬金 Buff を一括監視できます。
- **グループモード**：Buff 監視で「食物」「炼金」ショートカットボタンで一括設定できます。

### Buff 名が分からないとき、監視対象を素早く見つけるには？

Buff の設定名が乱雑なため、まず「すべて監視」をオンにすると、オーバーレイにすべての Buff が表示されます。実際の効果で名称を確認してから、正確な監視リストに追加してください。
