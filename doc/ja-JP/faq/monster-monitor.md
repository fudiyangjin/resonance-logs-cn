## モンスター Buff と ライブモニター の Buff の違いは？

- **ライブモニター**：**自分**の Buff、スキル CD など。「ゲームオーバーレイ」で表示
- **モンスターモニター**：**現在攻撃メインターゲット**の Buff、味方がそのターゲットに付与した Buff、ヘイトリスト。「モンスターオーバーレイ」で表示

両方の設定は独立。Boss ギミックを追うときは モンスターモニター を使用してください。

## オーバーレイにはどのモンスターの Buff が表示されますか？

キャラクターの**攻撃メインターゲット**に応じて自動切り替え。モンスターテンプレート ID をファイルで指定する必要はありません。

- 別のモンスターにターゲットを切り替えて攻撃すると、モンスター Buff エリアとヘイトも追従
- 旧版 `ExtraBuffMonitoredMonsters.json` は**廃止**。追加・削除しても効果なし

## ダミー / 雑魚に Buff が出ないのはなぜ？

よくある原因：

1. **ダミーが現在の攻撃ターゲットではない**：確認したいダミーにダメージを与えるか、ターゲットにしないとオーバーレイにデータは出ません
2. **モンスターモニターを有効化 未有効**：ページ上部 **モンスターモニターを有効化** がオンで、**モンスターオーバーレイの切替** によりオーバーレイを表示しているか確認
3. **表示する Buff が未設定**：**Buff 監視** タブで該当 Buff を「自身のみ」または「グローバル」に追加（または「本キャラが付与した Buff をすべて監視」で名称を探す）
4. **ターゲットが死亡**：モンスター死亡後は Buff / ヘイト更新が停止。新ターゲットへ切り替えるか次の対象を待つ

ダミー討伐時もダミーが攻撃ターゲットとしてロックされていれば、Boss と同様に動作します。
