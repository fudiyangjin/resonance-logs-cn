# 履歴

**DPSメーター → 履歴** に対応します。

履歴はリアルタイム統計より詳細な振り返りができ、例えば：

- **ターゲット別ダメージ内訳**：ターゲット（首領、雑魚など）ごとに各スキルのダメージを分割
- **スキル分布の詳細**：展開するとスキル下の複数ダメージソースを表示
- **戦闘全体の集計データ**：戦闘全体のサマリー統計を含む

![履歴の例](../../../shared/img/dps/dps_3.png)

- 履歴が 200 件を超えると、次回起動時に古い記録を時間順に削除しシーケンスをリセットします。履歴の手動メンテナンスは通常不要です。

ダミー討伐モードは **討伐終了** 時に今ラウンドの結果を履歴に書き込みます。日常の振り返りは凍結したライブ画面で十分な場合もあり、複数戦の比較時に履歴を開いてください。詳細は [DPS 概要 · ダミー討伐モード](./README.md#ダミー討伐モードの使い方) を参照。
