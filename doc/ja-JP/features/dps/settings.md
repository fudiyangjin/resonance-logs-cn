# 設定

**DPSメーター → 設定** に対応します。

## 設定項目

| 項目 | パス |
|------|------|
| プレイヤー列表示 | DPSメーター → 設定 → ライブ → DPS（プレイヤー）列 |
| スキル列表示 | DPSメーター → 設定 → ライブ → DPS（スキル詳細）列 |
| 更新頻度 | DPSメーター → 設定 → ライブ → 更新レート |
| DPS 数値の略記 | DPSメーター → 設定 → ライブ → DPS 値を短縮表示 |
| ダミー討伐ボタン | DPSメーター → テーマ → ライブ → ヘッダー設定 → 訓練用ダミーボタンを表示 |
| 活動戦闘時間 | DPSメーター → テーマ → ライブ → ヘッダー設定 → アクティブ戦闘時間を表示 |
| コンパクトモード | DPSメーター → テーマ → 一般 → コンパクトモード |
| タイトルバーレイアウト | DPSメーター → テーマ → ライブ → ヘッダー設定 |
| ボス HP | DPSメーター → テーマ → ライブ → ヘッダー設定 → ボス HP |
| パケットキャプチャ | DPSメーター → 設定 → ネットワーク |
| ショートカット | DPSメーター → 設定 → ショートカット |

外観の詳細は [テーマ](./themes.md)、ダミー討伐の流れは [DPS 概要](./README.md#ダミー討伐モードの使い方) を参照してください。
