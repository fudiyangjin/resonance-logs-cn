# テーマ

**DPSメーター → テーマ** に対応します。

DPS 画面は**高度にカスタマイズ**でき、外観とレイアウトを調整できます。

![テーマページ](../../../shared/img/dps/dps_4.png)

- **カラーテーマ**：暗色・亮色・ピンク・緑・抹茶・虹などのプリセット、または項目ごとのカスタム色
- **サイズとレイアウト**：テーブル、ヘッダー、フォント、余白など
- **タイマーとボタン**：**アクティブ戦闘時間を表示**、リセット / 一時停止 / Boss フィルターなど
- **列の表示と順序**：[設定](./settings.md) でプレイヤー / スキル列を切替

プリセットとカスタムを組み合わせて好みの DPS 表示にできます。

## 活発戦闘時間

- **秒間ダメージ (DPS)**：戦闘全体の経過時間で計算
- **真秒間ダメージ (TDPS)**：実際にダメージが発生した「活発時間」で計算

設定パス：**DPSメーター → テーマ → ライブ → ヘッダー設定 → アクティブ戦闘時間を表示**。

## コンパクトモード

**コンパクトモード** は Live DPS ウィンドウを核心データのみに絞ります（プレイヤー、職業、戦力、シーズン強度、総ダメージ、DPS、真 DPS、割合など）。

設定：**DPSメーター → テーマ → 一般 → コンパクトモード**。

総ダメージ・DPS・割合だけ見る場合は有効化を推奨。戦力とシーズン強度は別途設定が必要です。

![Live コンパクトモード例](../../../shared/img/dps/dps_5.png)

## タイトルバー自由レイアウト

タイマー、Boss HP、ボタンエリアなどの配置を自由に設定できます。

設定：**DPSメーター → テーマ → ライブ → ヘッダー設定**。

Boss HP の横向き表示など、狭いウィンドウ向けオプションもあります。

**訓練用ダミーボタンを表示** を有効にすると Live ヘッダーに照準ボタンが表示され、[ダミー討伐モード](./README.md#ダミー討伐モードの使い方) を切替できます。

![タイトルバー自由レイアウト例](../../../shared/img/dps/dps_6.png)

その他の設定パスは [設定](./settings.md) を参照してください。
