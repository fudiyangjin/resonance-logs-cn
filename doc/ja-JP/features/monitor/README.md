# Buff 監視

スキル CD、Buff 監視、カスタムパネルなどの機能説明です。

## 概要

ライブモニター は次の部分に分かれています。

- **スキル CD**：スキルクールダウンを監視
- **Buff 監視**：指定 Buff の持続時間、スタック数などを監視
- **キャラクターパネル**：攻撃速度、クリティカル率、知力などの属性を表示
- **カスタム監視**：通常監視エリア、因子エリア、カウンター、Buff プログレスバー
- **ウィンドウを有効化**：オーバーレイ各モジュールのオン/オフとレイアウト編集

> **⚠️ 途中でアプリを開く場合**：ゲーム進行中にアプリを起動すると、スキル CD、キャラクターパネル属性などが不完全に見えることがあります。ゲームは**差分更新**のみ送るため、アプリは現在の完全な状態を取得できません。**マップ切り替え**（シーン/マップの変更）を 1 回行い、全属性の同期後に表示が正常に戻ります。

## 有効化とプリセット

- **ライブモニターを有効化**：ページ上部のマスタースイッチ。オフにするとオーバーレイへデータを送信しません。
- トップバーの **オーバーレイウィンドウの切替** / **オーバーレイレイアウトを編集** は [ウィンドウを有効化](./overlay.md) を参照。

## プリセット

- 職業/プレイスタイルごとに異なる**プリセット**を作成可能
- 各プリセットでスキル CD、Buff 監視、レイアウトなどを独立設定
- プリセットを切り替えると監視設定一式が切り替わります

## ベストプラクティス

設定完了後は **`Ctrl+\`** で**UI 非表示モード**を使うことを推奨します。メイン画面が収納され、スキル CD、リソース、Buff などオーバーレイのみが表示されます。オーバーレイを画面端や視界の端に置けば、大画面でも視線を下げずにクールダウンとリソースを把握できます。
