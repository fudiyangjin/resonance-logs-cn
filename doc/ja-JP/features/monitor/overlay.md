# ウィンドウを有効化

**ライブモニター → ウィンドウを有効化** タブに対応します。

ゲーム内透明オーバーレイ（オーバーレイウィンドウ）各領域の表示を制御し、レイアウト編集と併用します。スイッチ状態は**プリセットごとに保存**。

## ページ上部の操作

ライブモニター ページのトップバー（任意のタブで表示）：

- **オーバーレイウィンドウの切替**：ゲーム内オーバーレイの表示/非表示
- **オーバーレイレイアウトを編集**：レイアウト編集モード。オーバーレイ上で各領域（各 **カスタム監視エリア** の独立領域を含む）をドラッグ、リサイズ

先に下記 **ライブモニターを有効化** マスタースイッチをオンにしないと、データはオーバーレイへ送信されません。

## オーバーレイモジュール

本ページで各領域の表示状態を切り替え（プリセットごとに保存）。一般的なモジュール：

- **スキル CD**
- **スキル持続時間**
- **リソース**
- **キャラクター属性エリア**（[キャラクターパネル](./panel-attr.md) 参照）
- **カスタムパネル**（[カスタム監視](./custom-panel.md) 参照）
- **シールド詳細**

## オーバーレイ位置

各モジュール（スキル CD、リソース、Buff パネル、キャラクターパネル、カスタムパネルなど）はドラッグで位置調整可能。

## 表示形式

- Buff 持続時間は「グループ」「時間」などの形式をサポート
- 名称、残り時間、スタック数の表示有無を設定可能

設定完了後、[概要 · ベストプラクティス](./README.md#ベストプラクティス) を参照し `Ctrl+\` UI 非表示モードでオーバーレイを視界の端に配置してください。
