# キャラクターパネル

**ライブモニター → キャラクターパネル** に対応します。

オーバーレイでキャラクター属性（攻撃速度、クリティカル率、知力、急速、幸運など）を表示。戦闘中にパネル数値と照合しやすくなります。

属性が不完全、または数値が明らかに遅延している場合は差分同期の問題が多い。[概要のマップ切り替え](../README.md#buff-監視) を 1 回行ってから再確認してください。

## 属性の有効化

- リストで監視する属性にチェック。上部に「有効 x / 合計」を表示
- 未チェックの属性はオーバーレイに表示されません

## 表示色

- 有効化した各属性に**表示色**を個別設定。オーバーレイでエントリを区別しやすくなります

## レイアウト

- **行間**、**フォントサイズ**、**名称と数値の間隔**：キャラクターパネルページ内で調整
- **行順**：有効化した属性のオーバーレイ上下順を調整（上へ / 下へ）

## オーバーレイ表示

**ライブモニター → ウィンドウを有効化** で **キャラクター属性エリア** の表示を切り替え。[ウィンドウを有効化](./overlay.md) の他モジュールと一緒にレイアウト可能。
