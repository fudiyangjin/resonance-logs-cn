# スキル CD

**ライブモニター → スキル CD** に対応します。

途中でアプリを開いて CD が不完全な場合は、先に [マップ切り替えで同期](../README.md#buff-監視)（概要の説明を参照）。

## 職業とスキル CD

- **職業選択**：現在のプリセットで監視する職業を切り替え。スキルリストは職業に応じて変化（[プリセット](../README.md#プリセット) に紐づく）
- **スキル選択**：最大 **10** スキルを選択（オーバーレイは約 2 行 × 5 列）。クールダウンと残り時間を表示。一括クリアして再選択可能

## 持続時間スキル

一部スキルは CD に加え**持続効果時間**（地面効果、チャネリングなど）があり、**持続時間スキル** で個別に選択可能。上記スキル CD リストとは独立設定。オーバーレイの **スキル持続エリア** に表示（[ウィンドウを有効化](./overlay.md) 参照）。

## 共鳴スキル

**共鳴スキル** は名称検索で追加する必要があり、職業表から自動列出されません。選択リストは個別管理でき、共鳴系特殊スキルの監視に適しています。

## 監視プレビュー

ページ内の **監視プレビュー** が現在の選択順でアイコン配置を表示。順序を調整してからゲーム内で確認しやすくなります。

## スキル変換

一部スキルは特定 Buff 発動時に別形態へ変化し、スキル CD 欄は現在の Buff 状態に応じてアイコンと名称を自動切り替えします。例：青嵐騎士の「飛鳥投」は特定 Buff 時に「極·嵐切」、氷法の「寒冰風暴」は詠唱免除 Buff 時に「詠唱免除の寒冰風暴」と表示。
