# Buff 監視（モンスター）

**モンスターモニター → Buff モニター** タブに対応。

**現在攻撃メインターゲット**に表示する Buff 種類を設定（実際に表示されるモンスターはゲーム内現在ターゲットで自動切り替え）。検索リストは **Boss Buff** 設定表向けで、ライブモニター のプレイヤー Buff リストとは異なります。

## Buff 検索と選択

検索前に追加方式を選択：

| モード | 意味 |
|------|------|
| **検索で自身のみ追加** | **自キャラ**がモンスターに付与した Buff のみ表示 |
| **検索でグローバル監視に追加** | 付与者に関わらず、モンスターに Buff があれば表示 |

検索ボックスにキーワード入力後、結果 grid から選択して対応リストへ追加。反対側に既存の Buff は状態表示（例：「現在グローバル」）。

## 自身のみ付与

- 自キャラがモンスターに付与した Buff のみ追跡
- **本キャラが付与した Buff をすべて監視**：オンにするとオーバーレイに自キャラ付与 Buff をすべて列出。ゲーム内名称対照後にリストを絞り込み可能。オン時は下の個別選択は無視可
- 選択 Buff はアイコン chips で表示。クリックで削除

## グローバル監視

- モンスターに Buff があれば表示。**付与者を区別しない**（Boss ギミック Buff、パーティ debuff など向け）
- 選択 Buff もクリックで削除可能

同一 Buff は「自身のみ」と「グローバル」の一方または両方に存在可（検索時に現在の所属を表示）。リストから削除し両方で監視しなくなると、**優先度**と**アラート**設定も同期削除。

## Buff 優先度

モンスター Buff が多くオーバーレイスペースが限られる場合、**表示順**を指定。リスト上位の Buff を優先表示。

先に上で「自身のみ」または「グローバル」監視へ追加し、検索で優先度リストへ追加。上へ/下へ/削除をサポート。

## Buff カウントダウンアラート

**監視中のモンスター Buff** に残り時間アラートを個別設定：

- 残り時間 **≤ 設定秒数** でハイライト（色設定、オプション点滅）
- 「自身のみ」または「グローバル」に既にある Buff のみ有効

## 表示名称

監視 Buff に**オーバーレイ表示エイリアス**を設定（左側はゲームデフォルト名）。監視リスト設定後に編集可能な項目が表示されます。

## テキストパネルスタイル（モンスター Buff エリア）

**モンスター Buff エリア**のオーバーレイ文字とプログレスバー外観を調整：

- 行間、列間、フォントサイズ
- 名称色、数値色、プログレスバーの色と透明度

味方 Buff エリア、ヘイトリストエリアのスタイルは [味方 Buff](./teammate.md)、[ヘイトリスト](./hate.md) で個別設定。
