# ヘイトリスト

**モンスターモニター → ヘイトリスト** タブに対応。

オーバーレイで**現在攻撃メインターゲット**の**ヘイト順**（モンスターのヘイトリスト内の誰が、相対順序）を表示。ターゲット切り替えでリストも切り替わり、タンクのヘイト状況や OT 判断に役立ちます。

## ヘイトリストの有効化

- **ヘイトリストを有効化** をオンにすると、ヘイトデータが解析されオーバーレイに表示
- 未有効の場合、[ウィンドウを有効化](./overlay.md) の **ヘイトリストエリア** スイッチは使用不可（先に本タブで有効化が必要）

## 表示人数

**最大表示キャラ数**：5–20 人で調整。オーバーレイのヘイトエントリ上限を制御（デフォルト約 5）。

## ヘイトパネルスタイル

**ヘイトリストエリア**のスタイルを個別設定（モンスター Buff エリア、味方 Buff エリアとは独立）：

- 行間、列間、フォントサイズ
- 名称 / 数値 / プログレスバーの色と透明度
