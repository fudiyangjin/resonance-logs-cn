# ウィンドウを有効化（モンスターオーバーレイ）

**モンスターモニター → ウィンドウを有効化** タブに対応。

**モンスターオーバーレイ**内 3 エリアの表示/非表示を個別制御（設定は保存され、再起動後も有効）。

## 表示エリア

| エリア | 説明 |
|------|------|
| **モンスター Buff エリア** | [Buff 監視](./buff.md) で設定した自身のみ / グローバル Buff |
| **味方 Buff エリア** | [味方 Buff](./teammate.md) で設定した Buff とカテゴリ |
| **ヘイトリストエリア** | [ヘイトリスト](./hate.md) のヘイト順（先にヘイトタブで有効化が必要） |

ボタンクリックで **表示 / 非表示** を切り替え。

## レイアウト編集

モンスターモニター ページトップバーで **モンスターレイアウトを編集** をクリック。ゲーム内オーバーレイ上で：

- 各エリア位置をドラッグ
- エリアサイズを調整

編集モードではオーバーレイ内 **位置をリセット**、**サイズをリセット**（画面に表示される場合）を使用可能。

トップバー **モンスターオーバーレイの切替** でオーバーレイ全体の表示/非表示。上記各区域スイッチと併用。
