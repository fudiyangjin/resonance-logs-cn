# 音声アナウンス

ツールボックスの **音声アナウンス** に対応します。オフラインで音色をクローン／微調整し、固定短句を管理し、Buff / ギミック発生時に**生成済み**音声を再生します（再生時はモデルを再読み込みしません）。

使い方は大きく 2 つに分かれます：

1. **共通の設定手順**：Buff / ギミックなどの各機能ページでイベントと文言を設定し、「いつ・何を」再生するかを決めます。
2. **アナウンス音声の生成**：音色ソース（ゼロショットクローン / 微調整）を選び、文言をローカル音声に合成します。

## 共通の設定手順

### 前提

1. **音声アナウンス → モデルと音色** で音声モデルをダウンロード／インストール（変換済みモデルの手動インポートも可）。
2. **概要と再生** で **音声アナウンスを有効化** をオンにし、音量・キュー方針・生成加速（自動 / CPU / Vulkan）を調整。

### Buff 画面でアナウンスを設定

最もよく使う **ライブモニター → Buff 監視** を例に、監視中 Buff の下にある **音声アナウンス** ブロックを展開してルールを追加します。

![Buff 画面でのアナウンス設定](../../../shared/img/voice/voice_1.png)

1. **監視中 Buff を検索して音声アナウンスを追加** をクリックし、対象 Buff をリストに追加。
2. トリガーイベントを選択：**付与時** / **残り x 秒** / **消失時**。
3. 各イベントの文言ソースを選択：
   - **自動生成**：Buff 名＋イベント種別から既定文言を生成（プレビュー可）
   - **カスタムテキスト**：手入力のアナウンス文（例：「ティナのCDが終わった」）。幻想関連 Buff では階級プレースホルダも利用可（後述）
   - **短句ライブラリ参照**：既存短句を再利用
4. **試聴** で確認。「未生成」と表示された場合は音声が未合成なので、後述の「アナウンス音声の生成」を参照してください。

> 右上にはこのブロックで設定済みのアナウンス数が表示され、**アナウンスを削除** でいつでも解除できます。

### カスタムテキストの幻想階級

一部の幻想由来 Buff は、元の幻想に **0～5 階** があります。アナウンスに階級を入れたい場合は、**付与時 / 消失時** のカスタムテキストにプレースホルダを書きます（「残り x 秒」では未対応）：

| プレースホルダ | 説明 |
|----------------|------|
| `${阶数}` | 中国語向け表記 |
| `${remodelLevel}` | 上記と同じ意味 |

例：カスタム「ティナ${阶数}が発効」→ 5 階のとき「ティナ5阶が発効」と読み上げ。

補足：

- 対応するのは **Buff 監視** と **モンスター Buff 監視** の **付与時 / 消失時** のみ。カウンター、Boss DBM、ダンジョンギミック、「残り x 秒」は階級展開しません。
- 設定すると **0～5 階それぞれ 1 本** と、階級不明時用のフォールバック（プレースホルダを除去した文）が必要に応じて生成されます。全 Buff を事前一括生成するわけではありません。
- **試聴** は例として 5 階で展開します。実戦では解決できた幻想階級の音声を選び、解決できなければフォールバックを使います。
- 文言変更後は **アナウンス一覧** で状態を確認し、**不足音声を一括生成** してください。

オーバーレイ上の Buff 名も、元幻想の階級が分かる場合は「名称 \| n階」と表示されます（音声設定とは独立）。

### その他の設定ルート

音声ルールはすべて各機能ページでインライン設定します（アナウンス一覧では新規作成しません）：

| 設定場所 | 主なイベント |
|----------|----------|
| ライブモニター → Buff 監視 | 付与時 / 消失時 / 残り x 秒 |
| ライブモニター → カスタム監視（カウンター） | 目標到達時 |
| モンスターモニター → Buff 監視 | 付与時 / 消失時 / 残り x 秒 |
| モンスターモニター → Boss DBM | 発動時 |
| ダンジョンギミック → ギミック音声 | 対応マップのギミック警告 |

### アナウンス一覧

**音声アナウンス → アナウンス一覧** は上記すべての設定を読み取り専用で集約します（Buff 監視、モンスター Buff、カウンター、Boss ギミック、ダンジョンギミックごとに表示）：

- **未生成** の場合は、先にモデル導入と音色選択を済ませ、**不足音声を一括生成**。
- 「〜へ移動」から元の設定ページへジャンプできます。

## アナウンス音声の生成

文言を設定したら、テキストを音声に合成する必要があります。音色ソースは **ゼロショットクローン**（手軽）と **微調整**（高品質）の 2 通りです。

### 方法 1：ゼロショット音色クローン

**短句ライブラリ → 一括生成** で **音色クローン** を選択。学習不要で、参照音声 1 本だけで使えます：

![クローン音色での一括生成](../../../shared/img/voice/voice_2.png)

1. **既存の音色を使う**（保存済みの音色プロファイルを再利用）、または **新しい音色を作成**：
   - 音色名を入力し、参照音声（WAV）を選択；
   - 必要に応じて **参照音声ファイルを保持** にチェック。
2. 生成したい短句を選択（全選択可）し、**生成開始**。生成後すぐに有効になります。

**参照音声のポイント**：**5〜10 秒のキャラクター乾き音（ドライボイス）** がおすすめです——単一話者、クリア、BGM なし、リバーブなし。参照がきれいなほど、クローンの再現度が上がります。

短句の状態：

- **生成済み**：再生可能
- **再生成が必要**：音色またはテキスト変更で古い音声が無効
- **未生成**：有効な音声がまだない

> 初回生成は時間がかかることがあります。完了後のトリガー再生はローカル音声のみで、モデルを再読み込みしません。

### 方法 2：微調整音色（上級）

ゼロショットクローンは数秒の参照音声に頼るため、類似度と安定性に限界があります。より高い再現度を求める場合は、対象キャラクターの音声データで専用モデルを **微調整** し、エクスポートしたフォルダを **モデルと音色** ページで **微調整音色** として参照します。

おおまかな流れ：

1. **素材の取得**：Bilibili 動画は [bilibilidownloadtool](https://github.com/NANblogink/bilibilidownloadtool) でダウンロード。
2. **音声抽出**：FFmpeg でロスレス PCM WAV に変換（元のサンプルレートを保持。先に MP3 化しない）。
3. **ボーカル分離**：[Ultimate Vocal Remover](https://github.com/Anjok07/ultimatevocalremovergui) で背景音を除去し、`Vocals` ステムを書き出して全編を確認。
4. **データセット作成**：無音区間で分割、24kHz にリサンプル、ASR で書き起こして人手で校正。
5. **学習**：音声離散コードと speaker embedding を生成し、Qwen3-TTS 0.6B CustomVoice を微調整（Docker 推奨、ビルド済みイメージ `ghcr.io/mozi1924/qwen3-tts-easyfinetuning`）。
6. **エクスポートと配備**：GGUF 配備フォルダをエクスポートし、アプリ内で微調整音色として参照。

データと環境の要件：

- 自分が所有する、または明示的に許諾された音声データのみ使用。1 データセットにつき話者は 1 人。
- クリーンな有効音声を **10〜30 分** 用意するのが理想。少量でも学習は可能ですが、類似度と安定性は下がります。
- 学習には NVIDIA GPU（VRAM 16GB 以上推奨）。Windows では WSL2 + Docker Desktop を推奨。

## 再生とキュー

- **音声アナウンスを有効化** がオフのときは、どのルールも再生しません。
- キュー方針：
  - **満杯時は低優先度を破棄**：混雑時の滞留を抑える
  - **高優先度は割り込み可能**：重要ギミックが再生を奪える
- いつでも **再生停止** できます。

## ロードアウトとの関係

総プリセットにはライブ監視・モンスター監視などの設定が含まれ、音声バインドも対応サブプリセットに保存されます。切替やインポート後は、アナウンス一覧で音声が **生成済み** か確認してください（別マシンでは再生成が必要な場合があります）。

## よくある質問

| 現象 | 対処 |
|------|----------|
| 試聴が失敗する | モデル導入済みか、短句／バインドの音声が生成済みかを確認 |
| バックエンドが「利用不可」 | Vulkan ランタイム導入後に **更新**。または CPU / 自動へ変更 |
| 微調整音色が使えない | 配備フォルダを再配置し、整合性チェックを実行 |
| ギミック項目が出ない | 現在のマップに音声ギミックが未接続 |
| 操作が競合する | ダウンロード／生成の完了を待つか、キャンセルして再試行 |
| `${阶数}` を入れたが階級が読まれない | イベントが「付与時 / 消失時」かつカスタム文言か確認；アナウンス一覧で各階バリアントを生成；一部の幻想 Buff は召喚体ではなくプレイヤー自身が付与するため、階級が一時的に解決できずフォールバックになることがあります |

マニフェスト／リリース詳細はリポジトリの `docs/VOICE_MODEL_MANIFEST.md`（開発者向け）を参照してください。
