# はじめに

初回利用時は次の手順に従ってください。詳細は [アプリ実行 FAQ](./faq/app-runtime.md#パケットキャプチャ) を参照してください。

## 1. 言語を選択

**DPSメーター → 設定 → 言語** を開き、**ゲームクライアントと同じ言語**（日本語・英語・中国語）を選んでください。Buff 名やスキル名が正しく表示されます。

## 2. Npcap をインストール

本アプリは Windows で **Npcap** を使ってパケットを取得します。先に [Npcap](https://npcap.com/) をインストールしてください（Windows 10 以降）。

## 3. ネットワークアダプタを選択

**DPSメーター → 設定 → ネットワーク** で、ゲーム通信が流れるアダプタを **ネットワークデバイス** から選びます。

- 実際の Ethernet / Wi-Fi アダプタを優先
- Loopback、VirtualBox、VMware などの仮想アダプタは避ける
- VPN やプロキシを一時的に無効にして試す

## 4. アプリを再起動

> **重要**：アダプタを変更した場合、または初めて選択した場合は、**アプリを完全に終了してから再起動**しないと新しい設定は反映されません。設定を保存しただけでは不十分で、以前のアダプタのまま、またはデータが表示されないことがあります。

## 5. ゲームを起動して確認

1. （アダプタ設定後）本アプリを再起動
2. ゲームを起動し、戦闘可能なシーンに入る
3. **DPSメーター** のライブウィンドウでダメージデータが出るか確認

データが出ない場合は [データが出ない / キャプチャできない](./faq/app-runtime.md#データが出ない--キャプチャできない) を参照してください。
