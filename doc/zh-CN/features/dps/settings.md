# 设置

对应 **DPS检测 → 设置**。

## 设置入口

| 设置项 | 路径 |
|--------|------|
| 玩家列显示 | DPS检测 → 设置 → 实时 → DPS（玩家）列 |
| 技能列显示 | DPS检测 → 设置 → 实时 → DPS（技能明细）列 |
| 刷新频率 | DPS检测 → 设置 → 实时 → 刷新频率 |
| 缩写 DPS 数值 | DPS检测 → 设置 → 实时 → 缩写 DPS 数值 |
| 打桩按钮 | DPS检测 → 主题 → 实时 → 标题栏设置 → 显示打桩按钮 |
| 活跃战斗时间 | DPS检测 → 主题 → 实时 → 标题栏设置 → 显示活跃战斗时间 |
| 紧凑模式 | DPS检测 → 主题 → 通用 → 紧凑模式 |
| 标题栏自由布局 | DPS检测 → 主题 → 实时 → 标题栏设置 |
| Boss 血量 | DPS检测 → 主题 → 实时 → 标题栏设置 → 首领血量 |
| 网络抓包 | DPS检测 → 设置 → 网络 |
| 快捷键 | DPS检测 → 设置 → 快捷键 |

主题相关外观说明见 [主题](./themes.md)。打桩流程见 [DPS 总览](./README.md#打桩模式使用说明)。
