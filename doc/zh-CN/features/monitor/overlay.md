# 启用窗口

对应 **实时监控 → 启用窗口** Tab。

控制游戏内透明浮窗（遮罩窗口）各区域的显示，并与布局编辑配合使用。开关状态**按方案保存**。

## 页面顶部操作

在实时监控页顶栏（任意 Tab 均可见）：

- **切换遮罩窗口**：显示或隐藏游戏内浮窗。
- **编辑遮罩布局**：进入布局编辑模式，在浮窗上拖动、缩放各区域（含各 **自定义监控区** 独立区域）。

需先在下方开启 **启用实时监控** 总开关，数据才会推送到浮窗。

## 浮窗模块

可在本页切换各区域的显示状态（按方案保存），常见模块包括：

- **技能 CD**
- **技能持续时间**
- **资源**
- **角色属性区**（见 [角色面板](./panel-attr.md)）
- **自定义面板**（见 [自定义面板](./custom-panel.md)）
- **护盾详情**

## 浮窗位置

各模块（技能 CD、资源、Buff 面板、角色面板、自定义面板等）均可拖动调整位置。

## 显示格式

- Buff 持续时间支持「分组」「小时」等格式
- 可设置是否显示名称、剩余时间、层数

配置完成后，可参考 [总览 · 最佳实践](./README.md#最佳实践) 使用 `Ctrl+\` 无 UI 模式，将浮窗置于视线余光处。
