# 角色面板

对应 **实时监控 → 角色面板**。

在浮窗中显示角色属性（如攻速、暴击率、智力、急速、幸运等），便于战斗中对照面板数值。

若属性显示不全或数值明显滞后，多为增量同步问题，请按 [总览说明切图一次](../README.md#buff-监控) 后再查看。

## 启用属性

- 在列表中勾选需要监控的属性；顶部会显示「已启用 x / 总数」
- 未勾选的属性不会出现在浮窗中

## 显示颜色

- 每个已启用属性可单独设置**显示颜色**，便于在浮窗中区分不同条目

## 布局

- **行间距**、**字体大小**、**名称与数值间距**：在角色面板页内调整
- **行顺序**：对已启用的属性调整浮窗中的上下排列顺序（上移 / 下移）

## 浮窗显示

在 **实时监控 → 启用窗口** 中可开关 **角色属性区** 的显示，与 [启用窗口](./overlay.md) 其他模块一并布局。
