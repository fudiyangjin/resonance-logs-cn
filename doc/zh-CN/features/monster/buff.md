# Buff 监控（怪物）

对应 **怪物监控 → Buff 监控** Tab。

用于配置**当前攻击主目标**身上要显示的 Buff 种类（实际显示哪只怪由游戏内当前目标决定，会自动切换）。搜索列表针对 **Boss Buff** 配置表，与实时监控的玩家 Buff 列表不同。

## Buff 搜索与选择

搜索前请先选择加入方式：

| 模式 | 含义 |
|------|------|
| **搜索加入仅自身施加** | 只显示**本角色**施加到怪物身上的 Buff |
| **搜索加入全局监控** | 不论谁施加，只要怪物身上有该 Buff 就显示 |

在搜索框输入关键词后，从结果 grid 点选即可加入对应列表；已存在于另一侧的 Buff 会显示状态提示（如「当前在全局」）。

## 仅自身施加

- 只追踪你打到怪物身上的 Buff。
- **监控本角色施加的全部 Buff**：开启后浮窗会列出你施加的所有 Buff，便于对照游戏内名称再精简列表；开启后下方单独勾选的条目可忽略。
- 已选 Buff 以图标 chips 展示，点击可移除。

## 全局监控

- 怪物身上出现即显示，**不区分**施加者（适合看 Boss 机制 Buff、团队 debuff 等）。
- 已选 Buff 同样可点击移除。

同一 Buff 只能归属于「仅自身」或「全局」其中一种；改选另一种模式时会自动切换归属。从监控中移除后，会同步清理其**优先级**、**告警**与**语音播报**配置。

## Buff 优先级

在怪物 Buff 较多、浮窗空间有限时，可指定**显示顺序**：列表靠前的 Buff 优先展示。

须先在上方加入「仅自身」或「全局」监控，再通过搜索添加到优先级列表，支持上移 / 下移 / 移除。

## Buff 倒计时告警

为**已监控的怪物 Buff** 单独设置剩余时间告警：

- 剩余时间 **≤ 设定秒数** 时高亮（可设颜色、可选闪烁）
- 仅对已在「仅自身」或「全局」中的 Buff 生效

## Buff 语音播报

在独立的「Buff 语音播报」区域搜索已监控 Buff，可分别配置生效、即将结束和失效播报。每项会标明「全局」或「仅自身」来源。

- **仅自身**严格按施加者实例判断；队友施加的同名 Buff 不会触发或延长你的播报。
- **全局**按 Buff 整体存在状态判断；多人施加时，第一份出现算生效，最后一份消失算失效，到期提醒使用最后一份实例的时间。
- 开启「监控本角色施加的全部 Buff」后可从完整 Buff 字典选择，但语音仍需逐个启用。

## 显示名称

为已加入监控的 Buff 设置**浮窗显示别名**（左侧为游戏默认名）。须先配置监控列表后才会出现可编辑项。

## 文字面板样式（怪物 Buff 区）

调整 **怪物 Buff 区** 在浮窗中的文字与进度条外观：

- 行间距、列间距、字号
- 名称颜色、数值颜色、进度条颜色与透明度

队友 Buff 区、仇恨列表区样式分别在 [队友 Buff](./teammate.md)、[仇恨列表](./hate.md) 中配置。
