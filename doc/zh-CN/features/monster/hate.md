# 仇恨列表

对应 **怪物监控 → 仇恨列表** Tab。

在浮窗中显示**当前攻击主目标**的**仇恨排序**（谁在怪物的仇恨列表中、相对顺序），目标切换时列表会随之切换，便于判断坦克仇恨与 OT 情况。

## 启用仇恨列表

- 打开 **启用仇恨列表** 后，仇恨数据才会解析并在浮窗中展示。
- 若未启用，[启用窗口](./overlay.md) 中的 **仇恨列表区** 开关不可用（需先在本 Tab 启用）。

## 显示人数

**最多显示角色数**：可调整 5–20 人，控制浮窗中展示的仇恨条目上限（默认约 5）。

## 仇恨面板样式

单独配置 **仇恨列表区** 样式（与怪物 Buff 区、队友 Buff 区独立）：

- 行间距、列间距、字号
- 名称 / 数值 / 进度条颜色与透明度
