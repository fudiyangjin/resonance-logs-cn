# 启用窗口（怪物遮罩）

对应 **怪物监控 → 启用窗口** Tab。

分别控制 **怪物遮罩** 浮窗内三个区域的显示与否（设置会保存，重启后仍生效）。

## 显示区域

| 区域 | 说明 |
|------|------|
| **怪物 Buff 区** | [Buff 监控](./buff.md) 配置的仅自身 / 全局 Buff |
| **队友 Buff 区** | [队友 Buff](./teammate.md) 配置的 Buff 与分类 |
| **仇恨列表区** | [仇恨列表](./hate.md) 中的仇恨排序（须先在仇恨 Tab 启用） |

点击按钮在 **显示 / 隐藏** 之间切换。

## 布局编辑

在怪物监控页顶栏点击 **编辑怪物布局**，可在游戏内浮窗上：

- 拖动各区域位置
- 调整区域大小

编辑模式下可使用浮窗内的 **重置位置**、**重置尺寸**（若界面提供）。

顶栏 **切换怪物遮罩** 用于整体显示或隐藏浮窗，与上述分区开关配合使用。
