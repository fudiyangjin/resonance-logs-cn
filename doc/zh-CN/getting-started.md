# 快速入门

首次使用请按以下顺序完成配置。完整说明见 [应用运行 FAQ](./faq/app-runtime.md#数据捕获)。

## 1. 安装 Npcap

应用通过 **Npcap** 在 Windows 上捕获网络包。请先安装 [Npcap](https://npcap.com/)（Win10 及以上）。

## 2. 选择网卡

打开 **DPS检测 → 设置 → 网络**，在 **网络设备** 中选择用于捕获游戏流量的网卡。

- 优先选择连接互联网的真实网卡（以太网 / Wi-Fi）
- 不要选择 Loopback、VirtualBox、VMware 等虚拟适配器
- 若使用 VPN 或代理，可先关闭后再试

## 3. 重启应用

> **重要**：更换网卡或首次选择网卡后，**必须完全退出并重新启动应用**，新网卡设置才会生效。仅保存设置而不重启，抓包仍可能使用旧设备或无数据。

## 4. （可选）切换界面语言

若游戏客户端为英文或日文，可在 **DPS检测 → 设置 → 语言** 中将界面语言改为与游戏一致，以便 Buff、技能名称正确显示。

## 5. 启动游戏并验证

1. 重新启动本应用（若刚完成网卡设置）
2. 启动游戏并进入可战斗场景
3. 打开 **DPS检测** 的实时窗口，确认是否出现伤害数据

若无数据，请参阅 [没有数据 / 抓不到包怎么办？](./faq/app-runtime.md#没有数据--抓不到包怎么办)。
