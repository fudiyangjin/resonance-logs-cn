# Changelog v0.1.9

## 更新内容

### DPS 相关

- 语言选择移动到主窗口左下角
- 切换窗口按钮根据窗口是否最小化变换状态
- reset windows 增加重置其他遮罩窗口位置

### 实时监控相关

- buff 搜索支持通过 buff id 进行检索 @dilone
- 对于每个功能区增加可选展示的黑色遮罩, 默认关闭
- 增加设置能够控制是否开启字体阴影
- 原有的字体设置增加关联实时监控栏中展示的字体(DPS检测-主题-通用-自定义字体-应用到悬浮窗)
- 增加惩击资源, 扩展 2202251 为特殊buff展示, 技能是否能够释放关联对应资源能量
- 切换窗口按钮根据窗口是否最小化变换状态

### 怪物监控相关

- 新增 N20 蒂娜降攻事件 CD
- 新增韧性条展示，和怪物 buff 和仇恨一样，展示当前主目标的（怪物监控 → 韧性条）
- 对于每个功能区增加可选展示的黑色遮罩, 默认关闭
- 增加设置能够控制是否开启字体阴影
- 原有的字体设置增加关联怪物监控栏中展示的字体(DPS检测-主题-通用-自定义字体-应用到悬浮窗)
- bossdbm 增加技能重命名设置
- 切换窗口按钮根据窗口是否最小化变换状态

### 副本监控相关

- 娜宝, 巨塔地图调整
- 根据实际攻击范围调整浪宽, 并做预受击判断进行展示(危险显示红色, 安全显示绿色)
- 支持可修改小地图上展示的实体大小
- 原有的字体设置增加关联实时副本监控中展示的字体(DPS检测-主题-通用-自定义字体-应用到悬浮窗)
- 切换窗口按钮根据窗口是否最小化变换状态

## 注意事项

### 兼容性提醒

- 如果之前使用过 0.0.2 或 0.0.3，第一次使用 0.0.4-0.1.9 需要清理 `%LOCALAPPDATA%\resonance-logs-cn` 目录下的 `resonance-logs-cn.db`，再重新启动应用
- 0.1.5 由于修改了实例存储 key（uid -> uuid）、抓包逻辑，整体改动较大，若有异常请及时联系并提供日志
- 0.1.5 新增的因子有些没有进行自测（因子太贵了，并且没时间），如果有异常请及时反馈

### 特别注意

- 如果你自己修改了代码又不想提交 PR，在分享给其他人的时候请把应用名、版本号等原版相关信息都改掉，不要带着原仓库的信息，尤其是应用名和版本号
- 关闭按钮现在会隐藏到右下角，可以将隐藏栏的内容拖到右下角显示，避免窗口程序过多时无法快速找到
- 常见问题见使用手册中的 HTML
